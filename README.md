## 运行环境需求

- 搭载已Root的Android系统的设备
- Android系统版本 ≥ 10, (SDK Version ≥ 29)

## 安装

安装方法请自行搜索，关键词：

- `Xposed` / `Lsposed`
- `模块安装`
